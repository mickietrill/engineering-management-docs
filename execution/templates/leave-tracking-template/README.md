# Leave Tracking Template

This template provides a simple and transparent way to track employee leave
across months and categories in small to mid-sized teams.

It is designed to work in:

- Google Sheets
- Excel
- Notion (CSV import)
- Any spreadsheet tool

---

## Purpose

The template helps:

- Track vacations, sick days, and other leave types
- Maintain visibility across the team
- Avoid conflicts and understaffing
- Support lightweight people operations

---

## Leave Types

| Code | Type       |
|-----:|------------|
| V    | Vacation   |
| H    | Half-day   |
| S    | Sick       |
| M    | Maternity  |
| C    | Casual     |
| U    | Unpaid     |

---

## How to Use

1. Download [`leave-tracking.csv`](leave-tracking.csv)  
2. Import into Google Sheets or Excel
3. Add team members in the Name column
4. Mark leave days using the codes above
5. (Optional) Use colors for better visibility

---

## Included Files

- [`leave-tracking.csv`](leave-tracking.csv) — empty template
- [`example-filled.csv`](example-filled.csv) — example with sample data

---

## Best Practices

- Keep one sheet per year
- Lock header rows
- Use data validation for leave codes
- Use conditional formatting for colors
- Assign an owner for updates

---

## When This Works Best

- Teams up to ~50 people
- Startups and scaleups
- Lightweight HR processes
- Engineering/product teams

---

## When You May Need a Tool Instead

- Very large organizations
- Complex approval workflows
- Payroll integrations

In such cases consider dedicated HR systems.
